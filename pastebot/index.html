The Modular Paste Bot

A bot with a singular focus; help communities provide support by managing uploaded text files.

No local copies are kept. All files remain hosted on the Discord CDN, with your privacy in mind.

Automatic Processing

As soon as a text file is uploaded, a message with two buttons is sent.

One contains a link to a formatted page, the other one to the raw file.

Formatted Text

The paste is automatically detected and beautified to make it easily readable.

For ease of access, links to download and view the raw file are located at the top of the screen.

View Raw File

If you need to look at the file without any fancy colours, we've got your back.

And if you ever want to go back to easily readable, there's a link at the top.

Invite Link

Interested? Invite ModularPasteBot to your server, and keep everyone's Downloads folder clean of random log files!
